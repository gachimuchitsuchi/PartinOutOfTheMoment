## ゲーム内容
「友達を笑わせる」というコンセプトで自分で描いた絵と自声を入れてでゲームを作成しました。  
「!!」が出現した瞬間にボタンを押し、早く押すことで敵を倒すゲームです。  
５つのステージがあり、途中ギミックも用意しただボタンを押すだけのゲームにならないよう工夫しました。
